Chyper Engine Instructions

Welcome to Chyper Engine // BETA! Here's how to use it:

Input

Enter your text in the "DROP YOUR SIGNAL HERE..." text area.

Cypher Style

Select a cypher style from the "Cypher Style" dropdown:

  • Cold War Cuts (rot13): Applies a ROT13 cypher.
  • Jammed Coms (reverse): Reverses the text.
  • Interceptor 2.0 (base64): Encodes the text using Base64.
  • Payload Prism (hex): Encodes the text using hexadecimal.
  • Radiowave Distort (zalgo): Adds Zalgo characters to distort the text.
  • Shortwave Pulse (morse): Encodes the text in Morse code.
  • Signal Inversion (atbash): Applies an Atbash cypher.
  • QR Code (qr): Generates a QR code from the text.

Chaos Layer

Select a chaos layer from the "Chaos Layer" dropdown to add additional obfuscation:

  • None: No additional chaos layer is applied.
  • ASCII Artifact: Converts the text to ASCII codes.
  • Binary Scream: Converts the text to binary.
  • Corrupted QR: Corrupts the generated QR code.

Decoder Mode

Check the "Decoder Mode" checkbox to attempt to decrypt the input using the selected Cypher Style and Chaos Layer. Accurate selection is crucial for successful decoding.

Buttons

  • GENERATE: Generates the cyphered or decoded text based on your selections.
  • PLAY: Plays the sonic echo of the input text, representing the text as a sequence of notes.
  • EXPORT .TXT: Downloads the output text as a .txt file.
  • EXTRACT SEQUENCE: Downloads the sonic sequence as a .txt file.

Stealth Mode

Check the "Stealth Mode" checkbox to apply a high-contrast, desaturated filter to the page, making it appear similar to a terminal.

Theme Modes

Select a theme mode to change the appearance of the Chyper Engine:

  • Default Mode: The standard Chyper Engine appearance.
  • Hacker Mode: A dark theme, similar to a hacker terminal. The app title is hidden in this mode.
  • Lite Mode: A simplified theme with a dark gray background and no background image.
  • Static Mode: A theme with a static GIF background.
  • Glitch Mode: A theme with a glitch GIF background.

Export Options

  • COPY: Copies the output text to the clipboard.
  • EXPORT QR .PNG: Exports the generated QR code as a .png image. Only available when using the QR Code cypher style or Corrupted QR chaos layer.
  • EXPORT ASCII .PNG: Exports the output text as an ASCII art .png image.

Auto-Copy

Check the "Auto-Copy" checkbox to automatically copy the output text to the clipboard after generating.

Share

Click the "SHARE" button to copy a URL to your clipboard that will load the Chyper Engine with your current input and settings.
